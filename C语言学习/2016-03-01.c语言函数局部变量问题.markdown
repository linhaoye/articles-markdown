局部变量的作用只在函数内部，在函数返回后，局部变量的内在已经释放了。因此，如果函数返回的是局部变量的值，
不涉及地址，程序不会出错。但是如果返回的是局部变量的地址(指针)的话，程序运行后会出错。

原因是函数只是把指复制后返回了，但是指针指向的内容已经被释放了，这样指针指向的内容就是不可预料的内容，
调用就出错。准确来说，`函数不能通过返回指向栈内存的指釜`。
